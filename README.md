# 🎓 College Placement Analytics Dashboard

## 📌 Project Overview

An end-to-end data analytics project designed to analyze placement trends among engineering students. The project explores how factors such as CGPA, internships, projects, and skill development influence placement outcomes and salary packages.

The analysis was performed using **Excel**, **MySQL**, and **Tableau** to demonstrate the complete data analytics workflow from data preparation to business insights.

---

## 🎯 Objectives

* Analyze overall placement performance.
* Identify branches with better placement outcomes.
* Study the relationship between internships and placements.
* Evaluate the impact of employability and skills scores.
* Build interactive dashboards for stakeholder decision-making.

---

## 🛠️ Tools Used

* **Microsoft Excel** – Data Cleaning, Feature Engineering, Pivot Tables, Dashboarding
* **MySQL** – Data Modeling, SQL Queries, Business Analysis
* **Tableau** – Interactive Dashboard Creation and Visualization

---

## 📂 Dataset Information

The dataset contains records of **1,000 engineering students** with information related to academics, skills, internships, projects, and placements.

### Key Attributes:

* Student ID
* Branch
* CGPA
* Skills Score
* Employability Score
* Internship Count
* Internship Category
* Package Offered (LPA)
* Placement Status

---

## 📊 Project Workflow

### Excel

* Cleaned and transformed placement data.
* Created Employability Score and Skills Score metrics.
* Performed analysis using Pivot Tables.
* Built an interactive Excel dashboard.

### SQL

* Imported and structured data in MySQL.
* Conducted branch-wise placement analysis.
* Analyzed salary trends across branches.
* Evaluated the impact of internships and skills on placements.
* Used JOINs, CASE statements, GROUP BY, and aggregate functions.

### Tableau

* Designed an interactive placement dashboard.
* Implemented filters for branch, gender, placement status, company, and CGPA category.
* Visualized branch-wise performance and package distribution.
* Highlighted key insights for decision-making.

---

## 📈 Key Insights

* Out of 1,000 students, **77.9% secured placements**.
* The **average package offered was ₹5.5 LPA**.
* The **highest package reached ₹39.06 LPA**.
* **Computer Science students secured the highest average package of ₹6.1 LPA**.
* Students with **internship experience demonstrated better placement outcomes** than those without internships.
